# D7S 2A vs Waves CLA-2A Null Test Plan

Microtarefa 27/31.

Objetivo: comparar comportamento aproximado do D7S 2A com Waves CLA-2A.

## Setup

- Mesmo áudio fonte.
- Mesmo gain staging.
- Modo Compress e Limit em testes separados.
- GR alvo: 1 dB, 3 dB e 5 dB.
- Peak Reduction e Gain pareados perceptualmente.

## Procedimento

1. Renderizar D7S 2A.
2. Renderizar Waves CLA-2A.
3. Alinhar nível e amostras.
4. Inverter polaridade de uma versão.
5. Medir residual.

## Critério de aceitação

- Residual aproximado menor que -25 dB em material musical.

## Observação

Este teste depende do plugin proprietário Waves CLA-2A e deve ser executado no sistema do usuário.
