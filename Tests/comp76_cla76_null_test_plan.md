# D7S 76 vs Waves CLA-76 Null Test Plan

Microtarefa 26/31.

Objetivo: comparar comportamento aproximado do D7S 76 com Waves CLA-76.

## Setup

- Mesmo áudio fonte.
- Mesmo gain staging.
- Ratio 4:1 e 8:1 em testes separados.
- GR alvo: 3 dB e 6 dB.
- Attack/release pareados perceptualmente.

## Procedimento

1. Renderizar D7S 76.
2. Renderizar Waves CLA-76.
3. Alinhar nível e amostras.
4. Inverter polaridade de uma versão.
5. Medir residual.

## Critério de aceitação

- Residual aproximado menor que -25 dB em material musical.

## Observação

Este teste depende do plugin proprietário Waves CLA-76 e deve ser executado no sistema do usuário.
